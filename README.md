### Prerequisites  
Ensure you have **Java JDK 8+** installed on your system.  

### Steps to Run the Application

   ```bash
   git clone https://github.com/patilvikas580/VChat.git

   ```

   Open a terminal in the project folder and run:  
   ```bash
   javac Server.java
   java Server
   ```

